LuaV

LuaV is a customized version of Lua designed for Vixem. It supports plugin hooks, sandboxed execution, and rollback-safe scripting for multiplayer games.

Other Features

  • Global Admin Tooling
  • Animated UI Stacks
  • Group & Economy APIs
  • Game Creation Framework
